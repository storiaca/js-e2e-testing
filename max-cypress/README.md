```bash
npx cypress open
```

```bash
npm run test:open
```

```bash
npx cypress run
```

## Instalacija tipova

```bash
npm install -D @types/cypress
```

## Testiranje iz komandne linije tip web citaca

```bash
npx cypress run --browser firefox
```
